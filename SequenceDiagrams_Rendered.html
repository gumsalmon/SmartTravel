Sơ đồ Sequence: Quản lý Thời gian nghe trung bình

sequenceDiagram autonumber actor User as Khách Du Lịch (App/Web) participant Client as HeriStep Client participant API as HeriStep.API (AnalyticsController) participant Worker as VisitProcessingWorker (Background) participant DB as SQL Server (StallVisits Table) actor Admin as Quản Trị Viên (Admin Web) Note over User, DB: GIAI ĐOẠN 1: GHI NHẬN THỜI GIAN NGHE User->>Client: Ấn nút "Nghe thuyết minh" Client->>Client: Bắt đầu tính giờ (Timer Start) User->>Client: Ấn "Dừng" hoặc chuyển trang Client->>Client: Tính tổng thời gian (giây) alt Nếu là Mobile App (Gửi qua Queue) Client->>API: Gửi POST /api/Listen/TrackVisit (StallId, Duration) API->>Worker: Đẩy thông tin vào Hàng đợi (VisitQueue) Worker->>DB: Lưu bản ghi mới vào bảng [StallVisits] else Nếu là Web (Gửi trực tiếp) Client->>API: Gửi POST /api/Listen/TrackVisit API->>DB: INSERT trực tiếp vào bảng [StallVisits] end Note over User, DB: GIAI ĐOẠN 2: TRUY XUẤT THỐNG KÊ TRUNG BÌNH Admin->>Admin: Truy cập trang Analytics Admin->>API: Gửi GET /api/Analytics/avg-listen-time Note right of API: SQL Logic: SELECT AVG(listen_duration_seconds) API->>DB: Query: AVG() theo StallId DB-->>API: Trả về kết quả (giây) API->>API: Format Giây -> Phút:Giây API-->>Admin: Trả về JSON (stallName, avgDuration) Admin->>Admin: Hiển thị lên biểu đồ
