1. Chức năng Xem Bản Đồ Nhiệt (Heatmap)

sequenceDiagram autonumber actor Admin as Quản trị viên participant Web as Web Admin (Frontend) participant API as Analytics API (Backend) participant DB as SQL Server (Database) Admin->>Web: Tích chọn "Hiện bản đồ nhiệt" Web->>API: GET /api/analytics/heatmap activate API API->>DB: SELECT Latitude, Longitude FROM TouristTrajectories activate DB DB-->>API: Danh sách hàng ngàn tọa độ deactivate DB API-->>Web: Trả về JSON (List of Coordinates) deactivate API Web->>Web: Render L.heatLayer (Leaflet.heat) Web-->>Admin: Hiển thị vùng mật độ đỏ/cam trên bản đồ

2. Chức năng Xem Tuyến Đi Ẩn Danh (Route Tracking)

sequenceDiagram autonumber actor Admin as Quản trị viên participant Web as Web Admin (Frontend) participant API as Analytics API (Backend) participant DB as SQL Server (Database) Admin->>Web: Nhập Device ID & Bấm "Vẽ Tuyến Đi" Web->>API: GET /api/analytics/route/{deviceId} activate API API->>DB: SELECT * FROM TouristTrajectories WHERE DeviceId = id ORDER BY RecordedAt activate DB DB-->>API: Danh sách tọa độ theo thời gian deactivate DB API-->>Web: Trả về JSON (Sorted List of Points) deactivate API Web->>Web: Vẽ L.polyline nối các điểm Web->>Web: Đánh dấu Start/End Marker Web-->>Admin: Hiển thị đường di chuyển của khách
